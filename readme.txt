== Installation ==
1. Unpack the `download-package`.
2. Upload the file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Configure the options under Admin Panel `Settings -> Google Adsense`.
5. Done and Ready.

== Frequently Asked Questions ==

= What if I want to disable adsense code on specific Page or Post? =
* Add custom field `crunchify_adsense` with value `no` to disable ads on specific Page/Post in WordPress `Edit Post/Page Panel`. Checkout Screenshot.
